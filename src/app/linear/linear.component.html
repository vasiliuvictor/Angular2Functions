Linear Congruential generator



Definitie

Formula matematică a sa este: X(n) = ( a * a + c ) mod m, unde X(0) este seed, a este multiplicatorul, c este constanta. Daca m se alege bine, acest generator are multiple avantaje: este stabil, trece cu bine numeroase teste statistice, se calculează rapid, cere memorie puţină, teoria care sta la baza este bine înţeleasă.

M-ul a fost ales 65535 care reprezinta 2 ^16
